使用說明

1. 用支援 CDC/USB-Serial 嘅近藤 USB Adapter (Dual USB Adapter HS / ICS USB Adapter HS) 連接電腦。
2. 瀏覽器需支援 Web Serial API（Chrome / Edge 桌面版）。
3. 按「連接」後喺瀏覽器彈出視窗揀返個 serial port。
4. 設定變更（ID/Baud/EEPROM）時，請確保只有一顆 servo 連接。
5. 位置控制／角度讀取喺 daisy-chain 多顆 servo 情況下亦可使用（用返 ID 分辨）。
6. 通訊格式：8-bit data, 1 stop bit, Even parity，半雙工。
ICS3.5/3.6 通訊協議根據近藤科學公開之 ICS3.5/3.6 ソフトウェアマニュアル（コマンドリファレンス）實作。使用前請確認 servo 電源供應充足，切勿只用 USB 供電驅動 servo。
