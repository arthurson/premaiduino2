
it is a fork from Pre-maiduino.

It is based on the following elements

1, Arduino IDE 2.3.8

2, Arduino Core for STM32 2.12.0
https://github.com/stm32duino/Arduino_Core_STM32/releases

3, ICS Library for Arduino ver.3
https://kondo-robot.com/faq/ics-library-a3

it supports http controller.
